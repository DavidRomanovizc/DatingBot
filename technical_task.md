## 1.1 Цель проекта

Система состоит из следующих основных функциональных блоков:

1. Регистрация, аутентификация и авторизация
2. Создание анкеты
3. Просмотр анкет
4. Настройка фильтров
5. Чат-рулетка
6. Мессенджер
